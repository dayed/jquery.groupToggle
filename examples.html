groupToggle

A jQuery plugin for checking/unchecking a group of checkboxes

Examples

1. Make a group toggle using filter

The simpliest way to create a toggle is to call groupToggle with a filter. Please note that the plugin always applies 'input:checkbox' selector, so you don't need to specify it in the filter.

$('#group_toggle1').groupToggle('.group1');
// The same as:
// $('#group_toggle1').groupToggle({filter: '.group1'});

2. Make a group toggle for checkboxes by specifying their parent

Another way to create a group toggle is to specify a parent element of checkboxes. Just pass parent selector or parent jQuery object as a value of 'groupParent' option.

    Check/uncheck all from group 2

$('#group_toggle2_1').groupToggle({groupParent: '#chk_group1'}); $('#group_toggle2_2').groupToggle({
    groupParent: $('#chk_group2'),
    toggleOnClass: 'toggleOn',
    toggleOffClass: 'toggleOff'
});

3. Plugin methods

Title
// Check the whole gorup
$('#table_select_all').groupToggle('check');

// Uncheck the whole group
$('#table_select_all').groupToggle('uncheck');

// Update the group and refresh the toggle state
$('#table_select_all').groupToggle('update');

Plugin options

Option Default Description
filter null optional. Specifies filter which is applied to selected checkbox inputs.
groupParent null optional. Specifies a parent for the group of checkboxes. All checkboxes will be selected only within specified parent. This parmeter can be either a css selector or a jQuery object.
toggleOffClass '' optional. Specifies a class which will be added to non-checkbox toggle element when it's unchecked.
toggleOnClass 'checked' optional. Specifies a class which will be added to non-checkbox toggle element when it's checked.

Plugin methods

Method Description
check Checks all group checkboxes.
uncheck Unchecks all group checkboxes.
update Updates the list of checkboxes in the group (see filter and groupParent options). Method also updates the toggle state according to the state of the whole group of checkboxes.
